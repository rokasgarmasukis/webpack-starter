# Webpack App Starter

Includes:

- Webpack configuration using html template
- Eslint config (airbnb-base)
- Webpack devServer
- Style and css loaders
- Mini css extract plugin (used instead of style-loader)
- Asset loading
- Babel configuration
- Plugins:
  - HtmlWebpackPlugin
  - MiniCssExtractPlugin
  - BundleAnalyzerPlugin
